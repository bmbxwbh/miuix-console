什么是 Miuix Console？
Miuix Console 是基于小米 Miuix 设计语言的 Web 控制台 UI 框架，零依赖，纯 CSS + Vanilla JS。
如何安装？
引入 miuix-console.js 即可一键加载所有组件，也可以按需引入单独的 CSS 和 JS 文件。
支持哪些浏览器？
Chrome 80+, Firefox 80+, Safari 14+, Edge 80+，移动端 iOS Safari 14+ 和 Android Chrome 80+。
