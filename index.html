设计原则

从 Miuix 源码直接移植的六大核心特征

◇
Smooth Corner
贝塞尔曲线连续曲率过渡，k=0.55。不是简单的 border-radius，而是从直线到圆弧的丝滑过渡。
▦
多层 Surface
5 级色阶系统：#161616 → #0C0C0C → #1C1C1C → #242424 → #2C2C2C，层层递进创造深度感。
↻
弹簧动效
cubic-bezier(0.34, 1.56, 0.64, 1) 弹簧曲线，仅用于 Switch、BottomNav、Sidebar 等交互组件。
☽
深色优先
默认深色主题，源自 HyperOS 的暗色设计哲学。支持 Light/Dark 双主题无缝切换。
◈
毛玻璃
TopBar、Sidebar、BottomNav 均采用 backdrop-filter 毛玻璃效果，营造通透的层次感。
○
零依赖
纯 CSS + Vanilla JS，无任何第三方依赖。MiSans 字体通过 CDN 加载，开箱即用。

快速开始

引入文件即可使用，无需构建工具

index.html
<!-- 1. 引入 MiSans 字体 --> <link href="https://cdn.jsdelivr.net/npm/misans@4/lib/Normal/MiSans-Regular.min.css" rel="stylesheet"> <!-- 2. 引入 Miuix 核心 --> <link rel="stylesheet" href="core/tokens.css"> <link rel="stylesheet" href="core/base.css"> <link rel="stylesheet" href="core/animations.css"> <!-- 3. 按需引入组件 CSS --> <link rel="stylesheet" href="components/card.css"> <link rel="stylesheet" href="components/button.css"> <!-- ... 更多组件 ... --> <!-- 4. 使用组件 --> <div class="mx-card mx-anim-card-in"> <div class="mx-card-head"> <span class="mx-card-title">Hello Miuix</span> </div> <div class="mx-card-body"> <button class="mx-btn mx-btn-primary">开始使用</button> </div> </div>
